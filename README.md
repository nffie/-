### Выполненное задание

Были реализованы программы для анализа блокчейна Биткоина на основе
[https://www.blockchain.com/ru/explorer/api/blockchain_api](https://www.blockchain.com/ru/explorer/api/blockchain_api), а именно:

1. Поиск в первых 100 блоках потраченных coinbase-транзакций и определение номеров этих блоков.
2. Определение для заданного блока транзакций с минимальным и максимальным отношением комиссии к сумме транзакции.
3. Анализ первых 10 блоков, сгенерированных в заданную пользователем дату, с вычислением:

   * количества транзакций;
   * увеличения размера блокчейна в байтах;
   * среднего времени генерации одного блока;
   * блока с наименьшим хешем;
   * средней комиссии в 5-м блоке;
   * адреса, сгенерировавшего наибольшее число блоков, и полученной им комиссии;
   * общего количества транзакций с участием этого адреса.
